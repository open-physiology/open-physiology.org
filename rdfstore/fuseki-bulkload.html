How to bulkload data in Fuseki

  • Download Apache Jena from here: https://jena.apache.org/download/index.cgi
  • Expand the .tar.gz file
  • Enter the resulting directory and go to the "bin" subdirectory
  • Run the tdbloader with syntax like the following:
    • ./tdbloader --loc=(TDB location) (location of file to bulkload)
    Here, "TDB location" is the location where you want the resulting triple store to be held. It should match the --loc used to run Fuseki.
